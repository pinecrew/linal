linear algebra library for Rust

### Usage
Add `linal` to your `Cargo.toml`, for example:
```toml
[dependencies]
linal = "0.2"
```

Or use the latest version from this repo:
```toml
[dependencies]
linal = { git = "https://github.com/pinecrew/linal" }
```
